交互式 HTML 原型，基于可信数据空间连接器现有设计规范改造。

GitHub Pages 部署后访问地址：
`https://<你的用户名>.github.io/wenzhou-connector-prototype/`

## 功能说明

本原型包含以下页面：

### 新增功能
- **资源目录**：展示监管平台同步的所有公共数据资源，支持按资源标识/名称/状态/时间筛选，新增「挂载数据源」点击跳转
- **资源目录详情**：区分三种类型（库表 / API接口 / 文件）分别展示不同字段
  - 库表：库表信息 + 数据字段表
  - API接口：接口信息 + 请求参数 + 返回参数
  - 文件：文件信息 + 文件字段描述
- **授权协议管理**：提供方视角（完整信息可见）/ 加工方视角（隐藏连接信息）
- **授权协议详情**：提供方可见完整连接信息，加工方仅可见字段描述

### 改造功能
- **资源库管理**：新增「来源」列，公共数据隐藏连接信息/下载按钮
- **交付管理**：公共数据产品自动锁定配置区，由监管平台自动分配
- **消息中心**：新增「公共数据授权运营」来源，消息来源对应功能模块可点击跳转

## 设计规范

完全遵循现有可信数据空间连接器设计规范：
- Umi 4 + React 18 + Ant Design 5 风格
- 左侧暗色侧边栏 + 顶部 Tab 页签
- 搜索筛选区域 + 表格列表 + 分页
- 标签样式、按钮样式、交互风格与原有产品保持一致
